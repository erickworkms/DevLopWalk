# DevLopWalk
Exemplo de um jogo integrando IKs em Unreal C++

Utiliza conceitos básicos de uso de IKs de controle para contado do personagem com o chão em Unreal C++. Todos os modelos 3D e animações são publicos e disponiveis no site do Mixamo da adobe. O projeto tem foco exclusivamente na programação e aplicação de conceitos de gameplay. Pode ser um guia de aprendizado para quem tiver interesse no básico para se utilizar IKs um jogo em Unreal C++.

Use o mouse e teclado para jogar. A tecla de espaço faz o personagem pular, ASWD para se movimentar,Mouse para controlar a camera. São 4 personagens, no menu de escolha cada um deles estara relacionado a um tipo de linguagem de programação.

Example of a game integrating IKs in Unreal C++

It uses basic concepts of using control IKs to connect the character with the ground in Unreal C++. All 3D models and animations are public and available on Adobe's Mixamo website. The project focuses exclusively on programming and applying gameplay concepts. It can be a learning guide for those interested in the basics of using IKs in a game in Unreal C++.

Use the mouse and keyboard to play. The space key makes the character jump, ASWD to move, Mouse to control the camera. There are 4 characters; in the selection menu, each of them is associated with a programming language type.

Link to download the executable: https://www.dropbox.com/scl/fi/ibg7k9uzl223p18mmrayc/DevLopWalk.zip?rlkey=0en0wzd1k8t3mnt35u69vckm8&dl=0



Unreal C++でIKを統合したゲームの例

これはUnreal C++でキャラクターを地面に接続するための制御IKの基本的な概念を使用しています。すべての3DモデルとアニメーションはAdobeのMixamoウェブサイトで公開されています。このプロジェクトはプログラミングとゲームプレイの概念の適用にのみ焦点を当てています。Unreal C++でIKを使用する基本に興味がある人にとっては、学習のガイドとなるでしょう。

マウスとキーボードを使用してプレイします。スペースキーでキャラクターがジャンプし、ASWDで移動し、マウスでカメラを制御します。選択メニューには4つのキャラクターがあり、それぞれがプログラミング言語のタイプに関連付けられています
